# seedchanger_with_username
changes to (evaisa's seedchanger)[https://steamcommunity.com/sharedfiles/filedetails/?id=2284931352], to enter username into Noita

To edit the seedchanger mod from Noita workshop, open C:\Program Files (x86)\Steam\steamapps\workshop\content\881100\2284931352
(Note: If you use a different drive for your Steam Library, navigate there instead of C:\Program Files (x86)\Steam)

Edit init.lua and settings.lua, and replace with the code in this repository.

Name seed will override the numerical seed.
